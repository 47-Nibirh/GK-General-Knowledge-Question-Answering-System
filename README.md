Simple GK Question Answering System using RNN

Developed a simple General Knowledge (GK) Question Answering System using a Simple RNN in PyTorch.
The model preprocesses text data, converts words into embeddings, and predicts answers for basic GK questions using sequence learning.
Implemented NLP preprocessing techniques including text cleaning, tokenization, vocabulary creation, numerical encoding, and padding.
This project demonstrates the fundamentals of Recurrent Neural Network for natural language processing tasks.
